TKINTER

Mi proyecto en tkinter utiliza la API mysql-connect para crear un programa que permite agregar, quitar y buscar objetos en una base de datos MySQL de manera sencilla. La interfaz gráfica de usuario creada con Tkinter facilita la interacción con la base de datos y ahorra tiempo en comparación con el uso de comandos SQL.
